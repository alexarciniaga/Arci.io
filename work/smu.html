Southern Methodist University(SMU) wanted a set of email and landing page templates that were compatible and interactible within their Pardot instance.

Part One: Email Templates

These templates utilized the MJML framework and are fully compliant with all modern email clients including:
Gmail
Outlook(both online and 2016+)
Apple Mail
and all associated viewports and browsers of the above listed

Part Two: Landing Page Template

The landing page template was part of the same campaign the program template was a part of. If users would engage with the program email, they were sent to this page to fill out a form that would begin their application. This page is editable within Pardot and was built from the ground up.

Technologies used include:
HTML5
CSS
SASS
Javascript
And code snippets from Pardot to enable template functionality for the end user in the platform
